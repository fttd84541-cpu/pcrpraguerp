# POLICIE ČESKÉ REPUBLIKY RP - INFORMAČNÍ SYSTÉM

Oficiální webový informační a administrační systém Policie ČR RP určený pro správu náboru, MDT systému a interní evidence uchazečů.

---

## 🏛️ O PROJEKTU

Tento projekt představuje simulovaný policejní informační systém inspirovaný reálnými procesy Policie České republiky.

Systém je určen pro roleplay účely a slouží jako:
- náborový systém pro nové příslušníky
- interní MDT databáze
- informační portál organizace
- administrativní policejní panel

---

## 🚓 HLAVNÍ FUNKCE

### 📋 Náborový systém
- online formulář uchazečů
- automatické ukládání do databáze
- bodové hodnocení přihlášek
- status: PŘIJAT / NEPŘIJAT

### 🖥 MDT SYSTÉM
- zabezpečený přístup (Firebase Auth)
- zobrazení všech uchazečů
- detailní informace o kandidátech
- administrace přihlášek

### 📊 Administrace
- přehled všech dat v systému
- možnost správy uchazečů
- kontrola přístupu

---

## 💻 TECHNOLOGIE

- HTML5
- CSS3
- JavaScript (ES6)
- Firebase Authentication
- Firebase Firestore
- GitHub Pages / Vercel / Firebase Hosting

---

## 📁 STRUKTURA PROJEKTU

- `index.html` – hlavní stránka systému
- `o-nas.html` – informace o organizaci
- `formular.html` – náborový formulář
- `login.html` – přihlášení do MDT
- `mdt.html` – policejní databáze
- `style.css` – vizuální styl systému
- `app.js` – odesílání náborových dat
- `mdt.js` – načítání databáze
- `firebaseConfig.js` – konfigurace Firebase

---

## 🔐 BEZPEČNOST

- Firebase Authentication pro přístup do MDT
- omezený přístup k databázi
- ochrana proti neoprávněnému vstupu
- oddělení veřejné a interní části systému
